# Estudo de PHP - Estruturas condicionais e lógica de descontos

Projeto pra praticar estruturas condicionais (`if`, `else if`, `else`) e operadores lógicos simulando um sistema de cálculo de frete e desconto em compras.

## Conceitos estudados

- **Estruturas condicionais** — `if`, `else if`, `else` pra controlar o fluxo do programa
- **Operadores lógicos** — `&&` (AND) pra combinar múltiplas condições (ex: cupom ativo E primeira compra)
- **Comparação** — `==` pra verificar igualdade, `!=` pra verificar diferença
- **Variáveis booleanas** — uso de `true` e `false` pra representar estados (cliente VIP, cupom ativo, etc.)
- **Cálculos condicionais** — aplicar descontos e regras de frete baseado em condições diferentes
- **Exibição condicional** — mostrar mensagens diferentes na tela conforme o resultado das condições

## Lógica do exercício

- Cliente VIP → frete grátis
- Compra acima de R$ 200 → frete pela metade
- Cupom ativo + primeira compra → 20% de desconto
- Cupom ativo + compra acima de R$ 300 → 10% de desconto
- Verificação final se houve desconto na compra e/ou no frete
