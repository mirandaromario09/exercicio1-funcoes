# Estudo de PHP - Estruturas condicionais e lógica de mensalidade

Projeto pra praticar estruturas condicionais (`if`, `else if`, `else`) e operadores lógicos simulando um sistema de cálculo de mensalidade com descontos e multas.

## Conceitos estudados

- **Estruturas condicionais** — `if`, `else if`, `else` pra controlar o fluxo do programa
- **Operadores lógicos** — `&&` (AND) pra combinar múltiplas condições (ex: cliente ativo E dias de atraso)
- **Comparação** — `==` pra verificar igualdade, `>` pra verificar se houve desconto/multa
- **Variáveis booleanas** — uso de `true` e `false` pra representar estados (cliente ativo, plano anual, cupom ativo)
- **Cálculos condicionais** — aplicar descontos e multas baseado em condições diferentes
- **Exibição condicional** — mostrar mensagens diferentes na tela conforme o resultado das condições

## Lógica do exercício

- Cliente inativo → mensalidade zerada
- Plano anual → 25% de desconto na mensalidade
- Cupom ativo → 15% de desconto na mensalidade
- Cliente ativo com dias de atraso → multa aplicada
- Verificação final se houve desconto e/ou multa
